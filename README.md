## Build & run

Requires .NET 8 SDK and Godot 4.3-mono.

```bash
# Build
dotnet build PrisonersDilemma.csproj

# Run full simulation (writes Output/)
dotnet run -- --run-sim

# Run unit tests
dotnet run -- --run-tests
```
